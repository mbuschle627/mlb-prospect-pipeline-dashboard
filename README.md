# Prospect Evaluation Dashboard

This dashboard was built to evaluate baseball prospects using tool grades, player information, public performance data, and written evaluation notes.

## Purpose

The goal of this project is to create a baseball operations style dashboard that helps organize and compare prospects by tools, performance indicators, projection, and overall player profile.

## Features

- Prospect search and player profiles
- Tool grades for position players
- Pitcher grades and arsenal information
- Player-specific notes
- Projection summaries
- Organized dashboard layout for scouting and player development use

## Tools Used

- HTML
- CSS
- JavaScript
- Public baseball data
- Player evaluation notes

## How to Run

Open the `index.html` file in a browser.

## Data Note

This project uses public or personally created evaluation data only. No private or proprietary team data is included.
